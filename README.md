# Todo List Application

Это frontend pet-проект, разработанный с использованием Vue.js и TypeScript. Цель проекта - создать полноценное приложение, максимально приближенное к реальному продукту, и пройти весь цикл разработки: от идеи и проектирования интерфейса до реализации, тестирования и деплоя.

Приложение построено с упором на чистый data-flow, предсказуемое управление состоянием и чёткое разделение ответственности между компонентами. Основное внимание уделялось типобезопасности, читаемости кода и поддерживаемости, а не быстрому получению результата.

Проект разрабатывался как практический опыт обучения. Вместо следования пошаговым туториалам большинство решений принималось самостоятельно, на основе официальной документации и экспериментов.

Приложение задеплоено и доступно онлайн.  
Live demo: https://todo-list-beta-lake-67.vercel.app

## Возможности

Приложение позволяет создавать, выполнять и удалять задачи. Пространства дают пользователю возможность разбивать задачи на определённые цели, которые можно также создавать, удалять и редактировать. Реализована фильтрация задач и их сортировка, интерфейс реактивно обновляется в зависимости от состояния приложения. UI спроектирован с упором на простоту, понятность и удобство использования с помощью Figma.

## Технологии

В качестве основного фреймворка используется Vue.js с TypeScript. Для управления состоянием применяется Pinia. Стилизация выполнена с помощью Tailwind CSS. Проект собирается и запускается через Vite. Для тестирования используется Vitest.

## Зачем был создан проект

Проект был создан для закрепления frontend-фундамента и получения практического опыта работы с типизированным кодом, состоянием приложения, взаимодействием компонентов и UI-логикой. Он отражает мой текущий подход к frontend-разработке и решению задач.
